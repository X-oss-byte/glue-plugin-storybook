# Getting Started

##

 The **Storybook** plugin adds **Storybook** to your project, allowing you to create documentation for your components. **Storybook** is an open-source tool for building UI components and documenting them in an interactive way. With this plugin, you can easily document your components and showcase them in various states and scenarios.
